# 🌍 WanderSplit

A modern travel planning application that helps users organize trips, manage budgets, create packing lists, track destinations, and collaborate with travel companions.

## ✨ Features

- 🗺️ Create and manage trips
- 💰 Budget planning and expense tracking
- ✅ Packing checklists
- �� Interactive maps and trip stops
- 🤝 Trip sharing and collaboration
- 📱 Mobile support with Capacitor
- ☁️ Cloud synchronization with Supabase
- 🎨 Smart trip covers

---

## 🛠️ Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Supabase
- Capacitor
- Leaflet
- OpenAI API
